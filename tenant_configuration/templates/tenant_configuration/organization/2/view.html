Now, we are going to set up tags to help you categorize your clients. These help you group your clients by a characteristic or program that you may offer. Tags will be a great help when it comes to searching for clients, analzying your data, and creating reports. The best part is that clients can have multiple tags, and tags can also be added or removed as often as needed. Not sure where to start? For example, you can use:

newcomers - this could help to designate clients that are new to the country and may require additional supports or services

spanish language - you could use any language as a tag here to indicate that the client (in this case) speaks Spanish

entrepreneurship 101 - if you were running a program called Entrepreneurship 101, then you could create this tag to designate someone as being part of a program.

Now, why don't you go ahead and create a few tags. You should add tags for any programs that you currently offer, or plan to offer in the next 3 months.
